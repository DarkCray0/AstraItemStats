# 📊 AstraItemStats

**AstraItemStats** is a lightweight and flexible **Paper** plugin that adds dynamic statistic trackers to items.

Each item can store its own stats - such as **damage dealt, distance walked, jumps, kills**, and more - which are updated **in real time** and displayed directly in the item lore.

## ✨ Features
- 📦 Stats stored directly on items
- 🔄 Real-time lore updates
- 🧩 Easy extension via separate stat config files
- 🧭 In-game menu to attach and remove trackers
- ⚡ Optimized and performance-friendly
- 🛠 Developer-friendly, event-based system

## 🎯 Goal
Designed to be simple for players and powerful for developers, allowing easy creation of custom item statistics without modifying core code.
